プライバシーポリシー

Amazon Wishlist Sale Picker  |  最終更新: 2026-04-28

収集するデータ

本拡張機能はいかなる個人情報・閲覧履歴・Amazon アカウント情報も収集・送信・保存しません。

動作の仕組み

本拡張機能は、ユーザーが開いている Amazon.co.jp のウィッシュリストページの DOM を読み取り、 セール中の商品をその場でフィルタリングします。処理はすべてブラウザ内で完結し、 外部サーバーへのデータ送信は行いません。

ストレージ

Chrome の storage API を使用して、割引率フィルターや価格帯フィルターなどの 設定値をローカルに保存します。この設定はユーザーのブラウザ内にのみ存在し、外部に送信されません。

第三者への提供

収集するデータが存在しないため、第三者への情報提供は一切行いません。

お問い合わせ

ご質問は GitHub Issues までお願いします。

Privacy Policy

Amazon Wishlist Sale Picker  |  Last updated: 2026-04-28

Data Collection

This extension does not collect, transmit, or store any personal information, browsing history, or Amazon account data.

How It Works

The extension reads the DOM of the Amazon.co.jp wishlist page currently open in your browser and filters sale items locally. All processing happens entirely within your browser. No data is sent to any external server.

Storage

The extension uses Chrome's storage API to save your filter preferences (discount rate, price range) locally in your browser. This data never leaves your device.

Third Parties

Since no data is collected, no information is shared with any third party.

Contact

For questions, please open an issue on GitHub Issues.
